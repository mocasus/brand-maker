# Copilot Style Logo Prompt

**Visual DNA:** Ribbon/wave shape · Gradient blue-cyan · Fluid modern.
**Tested on:** ChatGPT (DALL-E 3)

## The Prompt

```
A flat vector app icon in the visual style of Microsoft Copilot logo.

Composition:
- Square 1:1 aspect ratio, iOS app icon proportions
- Solid [BACKGROUND_COLOR] squircle canvas with 25% corner radius
- Centered ribbon/wave shape in [ICON_COLOR], occupying 60-70% of area

Shape design:
- Flowing ribbon that curves back on itself forming an infinity-like
  or figure-8 shape
- Smooth curves, no sharp corners
- 2D flat but suggests dimensional folding
- Alternative: single wave curve with rounded ends

Color palette:
- Ribbon: gradient from [ICON_COLOR_START] to [ICON_COLOR_END]
  (or solid [ICON_COLOR] if flat only)
- Background: [BACKGROUND_COLOR] solid

Style: fluid AI collaboration aesthetic, feels like intelligence
flowing, modern flat vector.

--no 3D, shadows, text, watermarks, sharp corners, geometric rigidity
```

## Variables

| Placeholder | Default |
|-------------|---------|
| `[ICON_COLOR_START]` | `#00A2FF` (cyan) |
| `[ICON_COLOR_END]` | `#0078D4` (blue) |
| `[BACKGROUND_COLOR]` | `#FFFFFF` |
