# Grok Style Logo Prompt

**Visual DNA:** Angular X shape · Monochrome black/white · Sharp geometric.
**Tested on:** ChatGPT (DALL-E 3)

## The Prompt

```
A flat vector app icon in the visual style of Grok/xAI logo.

Composition:
- Square 1:1 aspect ratio, iOS app icon proportions
- Solid [BACKGROUND_COLOR] squircle canvas with 25% corner radius
- Centered angular X-shape in [ICON_COLOR], occupying 60-70% of area

Shape design:
- Bold X made of two intersecting angular strokes
- Strokes have slight tapering (wider at ends, narrower at cross point)
- Sharp geometric edges, but not razor-sharp — subtle micro-rounding
- Symmetric intersection at center
- Alternative: two overlapping angular blades forming X

Color palette (strictly two-tone, monochrome):
- X shape: [ICON_COLOR] — solid flat
- Background: [BACKGROUND_COLOR] — solid, no gradient

Style: bold techy AI aesthetic, sharp geometric, edgy but clean,
flat vector modern, feels like unfiltered AI power.

--no gradients, 3D, shadows, text, watermarks, curved strokes,
softness, decorative elements
```

## Variables

| Placeholder | Grok Default |
|-------------|--------------|
| `[ICON_COLOR]` | `#FFFFFF` (white) |
| `[BACKGROUND_COLOR]` | `#000000` (black) |

## Colors

```
Black:  #000000
White:  #FFFFFF
Gray:   #1E1E1E (dark variant)
```

## Variants

- **Inverted:** black X on white bg
- **Colored:** add accent color to one stroke
- **Softer:** replace angular X with rounded X (less edgy)
