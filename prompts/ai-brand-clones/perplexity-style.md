# Perplexity Style Logo Prompt

**Visual DNA:** Rotating diamond/pinwheel · Teal + white · Minimal geometric.
**Tested on:** ChatGPT (DALL-E 3)

## The Prompt

```
A flat vector app icon in the visual style of Perplexity AI's logo.

Composition:
- Square 1:1 aspect ratio, iOS app icon proportions
- Solid [BACKGROUND_COLOR] squircle canvas with 25% corner radius
- Centered rotating diamond/pinwheel shape in [ICON_COLOR], occupying
  60-70% of the icon area

Shape design:
- Diamond/pinwheel made of 4 curved petals rotating around center
- Each petal is a comma-shape or teardrop curving outward
- Petals meet at a central hexagonal or circular hub
- Rotational symmetry (90-degree, feels like it's spinning)

Color palette (strictly two-tone):
- Pinwheel: [ICON_COLOR] — solid flat fill
- Background: [BACKGROUND_COLOR] — solid, no gradient

Style: minimal information-flow aesthetic, geometric precision,
feels like knowledge in motion, flat vector modern.

--no gradients, 3D, shadows, text, watermarks, realistic details
```

## Variables

| Placeholder | Perplexity Default |
|-------------|--------------------|
| `[ICON_COLOR]` | `#FFFFFF` (white) |
| `[BACKGROUND_COLOR]` | `#20B8CD` (teal) |

## Colors

```
Teal:    #20B8CD
Dark:    #0C2340
White:   #FAFAFA
```
