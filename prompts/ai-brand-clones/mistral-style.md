# Mistral Style Logo Prompt

**Visual DNA:** Layered pixelated grid · Warm gradient (yellow-orange-red) · Techy retro.
**Tested on:** ChatGPT (DALL-E 3)

## The Prompt

```
A flat vector app icon in the visual style of Mistral AI logo.

Composition:
- Square 1:1 aspect ratio, iOS app icon proportions
- Solid [BACKGROUND_COLOR] squircle canvas with 25% corner radius
- Centered geometric grid pattern occupying 65-75% of area

Shape design:
- Rectangular grid made of small square pixels/tiles
- Grid arranged in a rectangular/square boundary
- Pixels colored in a warm gradient (top: yellow, middle: orange,
  bottom: red) — like a heatmap or wind pattern
- Slight vertical/horizontal alignment (like a pixelated flag or
  wind currents)
- Alternative: single large layered rectangle with 3-4 color bands

Color palette (multi-color gradient allowed for Mistral authenticity):
- Grid pixels: [COLOR_1] to [COLOR_2] to [COLOR_3] gradient
- Background: [BACKGROUND_COLOR] solid

Style: techy retro pixel aesthetic, warm palette, geometric precision,
feels like data visualization made beautiful.

--no 3D, shadows, text, watermarks, curved shapes, softness,
photorealistic details
```

## Variables

| Placeholder | Mistral Default |
|-------------|-----------------|
| `[COLOR_1]` | `#FFD700` (gold/yellow) |
| `[COLOR_2]` | `#FF8C00` (orange) |
| `[COLOR_3]` | `#DC143C` (crimson) |
| `[BACKGROUND_COLOR]` | `#FAFAFA` |
