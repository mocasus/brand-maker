# Kawaii Robot Icon Prompt

**Style:** Rounded rectangle robot head, tech-friendly, AI/SaaS branding.
**Best for:** AI apps, developer tools, tech startups.

## The Prompt

```
A cute and minimalist flat app icon featuring ONLY the head silhouette
of a friendly robot, viewed from the front. The head is a soft rounded
rectangle (squircle) with 60-70% corner radius, slightly wider than
tall (aspect ratio ~1.1:1).

Robot features:
- Two oversized vertical pill-shaped eyes, cut out as transparent
  negative space (background color shows through)
- Eyes occupy 30% of face height, symmetric spacing
- Small rectangular ear pieces on both sides (5% width, mid-height)
- Optional: single thin vertical antenna on top (5% of head height)
  with small [ICON_COLOR] circle at tip
- No mouth, no nose

Color palette (strictly two-tone):
- Robot body: [ICON_COLOR] — solid flat fill
- Background: [BACKGROUND_COLOR] squircle with 25-30% corner radius

Style: kawaii chibi robot, ultra-friendly AI mascot, modern flat vector,
Apple-esque simplicity meets Japanese cuteness. Perfectly centered and
symmetrical. Sharp vector edges. iOS 2024 app icon aesthetic. Readable
from 16px to 1024px.

--no body, arms, legs, wires, screws, complex mechanical details,
3D effects, shading, gradients, shadows, textures, outlines, text,
watermarks, multiple robots, side view
```

## Variables

| Placeholder | Description | Example |
|-------------|-------------|---------|
| `[ICON_COLOR]` | Robot fill | `#FFFFFF`, `#E8E8E8` |
| `[BACKGROUND_COLOR]` | Squircle bg | `#C15F3C`, `#6366F1` |

## Example Fill (Moyzell)

```
[ICON_COLOR]        = #FFFFFF
[BACKGROUND_COLOR]  = #C15F3C
```

Result: warm terracotta squircle with white robot head, big pill eyes,
small side ears, minimal antenna.

## Variants

```
+ small circuit-board pattern hint in center of forehead (very subtle,
  same color as body but 15% opacity)
```

**Sleepy variant** — replace eye spec:
```
- Horizontal short pill eyes at 15% face height (half-closed feel)
```
